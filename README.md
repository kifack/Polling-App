# Polling App
This Application was build using :
- NodeJS + Express
- MongoDB + Mongoose
- React

  To Run the project
  1. Fill your .env-local  et rename it .env
  2. Go to backend folder and run your server > npm run start
  3. Go to polling-app and run > npm run dev
